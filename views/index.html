Planetary demo website

What is this?

Planetary is a self-hosted, feature-rich ShareX server written in Node.js. Its main focus is content management through the dashboard.

Main features:

  • Thumbnails: video and soundfiles have smaller, 10 second clips created for them. PDFs and images are supported as well
  • Portable: Planetary uses SQLite, so if you want to move it to a different server, just tar it!
  • Live updating dashboard: No need to refresh the dashboard when you upload a new file
  • Experimental PWA: The dashboard can be installed as an app and used when you're offline
